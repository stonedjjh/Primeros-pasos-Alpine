Directiva: x-bind

En este ejemplo veremos el uso de la directiva x-bind (shorthand :) para enlazar propiedades de un elemento a datos de Alpine.js.

Descripción general: la directiva x-bind (o ':') vincula atributos o propiedades DOM a valores reactivos definidos en x-data, permitiendo que la vista se actualice automáticamente cuando cambian los datos.

Casos de uso mostrados

  1. :value — Enlace de valor: asigna dinámicamente el valor del input desde x-data.
  2. disabled — Enlace de propiedad: permite activar/desactivar controles según estado reactivo.
  3. :style — Enlace al estilo: permite enlazar estilos (p. ej. color) desde datos reactivos.

Notas: x-bind puede usarse para atributos o propiedades; las expresiones deben existir en x-data para ser reactivas.
