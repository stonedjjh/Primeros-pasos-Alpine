Uso de x-data y x-text: el body recibe { message: 'I ❤️ Alpine' } y en el h1 asignamos la variable message mediante x-text.

Uso de x-on o @

Aquí x-data recibe "{ contador: 0 }" y en el botón controlamos el evento de Alpine con @click, donde le sumamos uno al contador al ser presionado; el <span> muestra el valor de contador con x-text.

Uso de x-for

En este div bordeado se inicializa un estado con la propiedad Lenguajes (array con C++, Java, Python, PHP), una cadena otroLenguaje y una función agregarLenguaje que añade el valor del input al arreglo y limpia el campo. Se usa x-for para recorrer Lenguajes y mostrar cada elemento en la lista.
